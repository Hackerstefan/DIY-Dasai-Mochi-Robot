# DIY-Dasai-Mochi-Robot
i built a dasai mochi robot with an esp32-s3 😏

Getting started 😋
You got to have:
- an esp32-s3
- a 3.7V battery
- a mini swich
- an old speaker (from a smartphone)
- a charging controller

How to assemble 😜

Flashing:
Once you put the robot together, go  <a href="https://themochi.huykhong.com/">here</a> and flash to your esp32
(remember that you can use this as a smart watch, so to view notifications on it, open bluetooth, connect to THE MOCHI, and install the cronos watch app.
